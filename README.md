# C Programming Projects 🚀

This repository is a collection of my personal projects and exercises as I learn the C programming language. It serves as my practice ground for understanding memory management, file I/O, pointers, and command-line interfaces.

## 📂 Projects Included

* **`file_encode_decode`**: A command-line tool that uses a Caesar cipher to safely encrypt and decrypt text files using file stream processing (`fopen`, `fgetc`, `fputc`).
* **`tic_tac_toe`**: A terminal-based Tic-Tac-Toe game exploring 2D arrays, game loops, and basic game logic.
* **`Expense_Tracker`**: This is a simple expense tracker, takes input from user stores in a text file and also can print it on user command and also can calculate total expense.

## 🛠️ Tools Used
* Standard C Library (`stdio.h`, `string.h`, etc.)
* GCC Compiler
* WSL / Linux Terminal
